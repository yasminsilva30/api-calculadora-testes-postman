# API Calculadora e Produtos

Projeto desenvolvido em Node.js e Express para testes de API utilizando Postman.

## Funcionalidades

- Divisão
- Multiplicação
- Tratamento de divisão por zero
- Cadastro de usuários
- Listagem de produtos
- Filtro por categoria
- Tratamento de erros 404

## Tecnologias

- Node.js
- Express
- Postman

## Como executar

```bash
npm install
node server.js
```

API disponível em:

```bash
http://localhost:3000
```
